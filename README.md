# C_file_compile
Compiling  C File using GCC 


***Compiling  C File using GCC ***

```
gcc filename.c -o filename 

```

***Examples***

```
gcc 10.c -o exploit

```

***Execute the Exploit*** 

```
./exploit

```

***Final ***

```

./exploit -b 0 -v RemoteIP
```
